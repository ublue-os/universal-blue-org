Custom Operating System images based on Fedora Atomic

Universal Blue is a collection of container images using Fedora's support for OCI/Docker containers as a transport and delivery mechanism for operating system content. That's nerdspeak for the ultimate Linux client: the reliability of a Chromebook, but with the flexibility and power of a traditional Linux desktop. At long last, we've ascended.
